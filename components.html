SINAPSE Design System

Sistema de design completo para aplicações de monitoramento preditivo

Visão Geral do Sistema

Bem-vindo ao SINAPSE Design System

Este sistema de design foi desenvolvido especificamente para aplicações de monitoramento preditivo de redes FTTH. Baseado no protótipo completo do SINAPSE, oferece todos os componentes necessários para construir dashboards profissionais.

Implementação para Flask

Estrutura de Templates Flask

Como organizar o Design System para uso com Flask e Jinja2.

Estrutura de Pastas:

flask_sinapse/
├── app.py
├── requirements.txt
├── static/
│   ├── css/
│   │   └── sinapse.css      # Este Design System
│   ├── js/
│   │   ├── components.js    # Componentes JS
│   │   └── dashboard.js     # Lógica específica
│   └── images/
│       └── logo.svg
└── templates/
    ├── base.html            # Template base
    ├── dashboard.html       # Dashboard principal
    ├── devices.html         # Gerenciamento
    ├── alerts.html          # Sistema de alertas
    ├── analysis.html        # Análise preditiva
    ├── settings.html        # Configurações
    └── components/          # Componentes Jinja
        ├── kpi_card.html
        ├── alert_item.html
        └── device_card.html

<!-- templates/base.html -->
<!DOCTYPE html>
<html lang="pt-BR">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>SINAPSE - {% block title %}{% endblock %}</title>
    <link rel="stylesheet" href="{{ url_for('static', filename='css/sinapse.css') }}">
    <link rel="stylesheet" href="https://cdnjs.cloudflare.com/ajax/libs/font-awesome/6.4.0/css/all.min.css">
{% block head %}{% endblock %}
</head>
<body>
    <div class="app-container">
        {% include 'components/sidebar.html' %}
        
        <main class="main-content">
            {% block content %}{% endblock %}
        </main>
    </div>
    
    <script src="{{ url_for('static', filename='js/components.js') }}"></script>
    {% block scripts %}{% endblock %}
</body>
</html>

Macros Jinja para Componentes:

{# templates/components/kpi_card.html #}
{% macro kpi_card(title, value, icon, trend_value, trend_type='up') %}
<div class="kpi-card">
    <div class="kpi-header">
        <div class="kpi-title">{{ title }}</div>
        <i class="fas fa-{{ icon }} kpi-icon"></i>
    </div>
    <div class="kpi-value">{{ value }}</div>
    <div class="kpi-trend trend-{{ trend_type }}">
        <i class="fas fa-arrow-{{ trend_type }}"></i>
        <span>{{ trend_value }}</span>
    </div>
</div>
{% endmacro %}

{# Uso no template #}
{{ kpi_card('Disponibilidade', '99.87%', 'signal', '+0.12% (24h)') }}
